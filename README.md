# AI-Driven Cybersecurity Governance & Policy Decision Platform

This project builds an AI-powered system that transforms cybersecurity data into
policy-level decisions using machine learning, explainable AI, and MLOps.

## Core Capabilities
- Cyber Risk Index computation
- Policy compliance and violation detection
- Policy impact simulation (what-if analysis)
- Explainable AI for governance decisions
- GPU-accelerated training
- Continuous retraining using MLOps

## Tech Stack
- Python 3.10
- PyTorch (CUDA)
- XGBoost (GPU)
- MLflow
- FastAPI
- Streamlit

## Focus
Bridging cybersecurity operations and policy-making through responsible AI.
